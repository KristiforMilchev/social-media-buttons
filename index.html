SOCIAL MEDIA BUTTONS

A CSS library of easy customizable social media buttons.

  • How to use
  • Types of buttons

How to use

  1. Download the social-media-buttons.min.css file and include it in your HTML file.
  2. Add the associated class name to the button you want to style followed by the desired social network.

    • Rounded button
